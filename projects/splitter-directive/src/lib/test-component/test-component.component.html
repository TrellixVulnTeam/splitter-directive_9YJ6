EXAMPLES USING SPLITTER


    <style>
      {{".mainbox{
            width:500px;
            height:200px;
        }
        .boxmenu{
            width:20%;
            border:solid black 1px;

        }
        .box {
            width:60%;
            border:solid black 1px;
        }"}}
    </style>
    <div splitterDirective class="mainbox" >
        <div class="boxmenu"></div>
        <div class="box"></div>
        <div class="boxmenu"></div>
    </div>

In this example we can see a basic use in horizontal mode.

With class .mainbox we stablished the main size of the container of divs what we must have to define for a good function of the splitterDirective

With class .box and .boxmenu we stablished the initial width of the each div. 




        <style>
           {{" .box4 {
                width:25%;
                border:solid black 1px;
            }
            "}}
        </style>
        <div splitterDirective [addstyle]="'border-left-color:black;border-left-width:10px'" 
        [addhoverstyle]="'border-left-color:yellow;border-left-width:10px'" [size]="'200px'" >
            <div class="box4"></div>
            <div class="box4"></div>
            <div class="box4"></div>
            <div class="box4"></div>
        </div>

In this example we use propery size to limit the height on horizontal splitter mode.

The property addstyle and addhoverstyle we add custom styles for the splits. This way we can customize our splits or splits

With class .box4 We define the initial width of each div 




        <style>
           {{".mainboxV{
                height:400px;
            }
            .boxv {
                height:33%;
                border:solid black 1px;
            }
            .boxvmenu{
                height: 100px;
                border:solid black 1px;
            
            }
            "}}
        </style>
        <div splitterDirective [vertical]="true" class="mainboxV"  [size]="'400px'">
            <div class="boxv"></div>
            <div class="boxv"></div>
            <div class="boxv"></div>
        </div>

In this example we split in vertical mode setting a size of 600px that in vertical case is the width.

Class .mainboxV set the style of the main container

Class .boxvmenu and boxv we set the initial height and style of the splits
